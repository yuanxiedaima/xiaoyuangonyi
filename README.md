#### 介绍
校园公益系统--这是我的毕设作品，客户端有app端和浏览器端，数据库使用mysql

#### 软件架构
- Java+Android+SpringBoot+Vue+Element+MyBatis+Mybatis-plus+MySQL
- 数据库：使用MySQL存储数据
- 后台管理系统：主要是SpringBoot+Vue，B/S架构
- 手机客户端：Android studio 进行原生开发
